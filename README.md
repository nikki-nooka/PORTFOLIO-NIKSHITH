## рџ“ќ Customization

### Update Personal Information
- Edit component files to update:
  - Experience list in `components/Experience.tsx`
  - Skills and certifications in `components/Skills.tsx`
  - Projects in `components/Projects.tsx`
  - Contact information in `components/Contact.tsx`

### Styling
- Modify Tailwind colors in `tailwind.config.ts`
- Update global styles in `app/globals.css`

### Add Profile Picture
- Place your image in the `public` folder
- Update the About section in `components/About.tsx`
